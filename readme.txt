Asteroids 02 is easy retro style python game.

You need install for run:
- Python3
- Pyglet

Game controls:
- Up arrow: Forward
- Down arrow: Backward
- Right arrow: Turning right
- Left arrow: Turning Left
- Space: Shooting
- R: restart game
- D: Leave game (show gameover screen)
- F11: Enter fullscreen
- F12: Leave fullscreen
- Esc: Close window
